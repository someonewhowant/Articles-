# Pocket CLI — Минималистичный менеджер закладок в терминале

Этот проект представляет собой легковесную утилиту для управления закладками прямо из командной строки. Главная особенность — полное отсутствие внешних зависимостей (zero dependencies), использование только встроенных модулей Node.js.

## Особенности
- **Интерактивный режим (REPL)**: Работайте со списком закладок без перезапуска приложения.
- **Поддержка тегов**: Организуйте свои ссылки с помощью меток.
- **Поиск**: Быстрый поиск по заголовкам, URL и тегам.
- **Открытие в браузере**: Мгновенный переход по ссылке системной командой.
- **Красивый интерфейс**: Табличный вывод и цветовая индикация с использованием ANSI-кодов.
- **Локальное хранение**: Данные сохраняются в JSON-файл в папке проекта.

## Установка и запуск

1. Убедитесь, что у вас установлен **Node.js** (версии 12 и выше).
2. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd articles/js/pocketcli
   ```
3. (Опционально) Сделайте файл исполняемым:
   ```bash
   chmod +x pocket.js
   ```

### Запуск в интерактивном режиме:
```bash
node pocket.js
```

### Выполнение разовой команды:
```bash
node pocket.js list
```

## Команды

| Команда | Описание | Пример |
| :--- | :--- | :--- |
| `add` | Добавить новую закладку | `add "https://github.com" "GitHub" --tags=dev,git` |
| `list` | Показать все закладки | `list` |
| `search` | Поиск по ключевому слову | `search "dev"` |
| `show` | Просмотр деталей (по ID) | `show 1` |
| `edit` | Редактировать поле | `edit 1 tags "news,tech"` |
| `open` | Открыть в браузере | `open 1` |
| `delete` | Удалить закладку | `delete 1` |
| `help` | Справка | `help` |

## Технические детали

Проект разработан как демонстрация возможностей стандартной библиотеки Node.js. 

- **Readline**: Модуль для реализации интерактивного ввода.
- **FS (File System)**: Синхронное чтение и запись в JSON-файл для персистентности.
- **Child Process**: Использование `exec` для интеграции с операционной системой (открытие браузера).
- **ANSI Escape Codes**: Ручное управление цветами терминала без сторонних библиотек.

Подробный разбор кода и архитектурных решений можно найти в файле [**clijs.md**](js/pocketcli/clijs.md).
