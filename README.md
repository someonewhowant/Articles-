# Сборник статей и проектов

Данный репозиторий представляет собой базу статей, обучающих материалов и практических примеров реализации различных инструментов. Основной упор сделан на создание туториалов и заметок для описания несложных приложений с использованием минимального количества внешних зависимостей.

## Навигация по материалам

### CLI Инструменты
*   **[Создание Pocket CLI на Node.js](js/pocketcli/clijs.md)** — Подробная статья о разработке менеджера закладок для терминала. Разбираются темы работы с `readline`, файловой системой и ANSI-цветами.
    *   [Просмотр исходного кода](js/pocketcli/pocket.js) — Основной скрипт приложения.

### JavaScript и основы
*   **[Реализация методов массива в JavaScript с нуля](js/arrays.md)** — Глубокое погружение во внутреннее устройство массивов. Разбор реализации `map`, `filter`, `reduce` и других методов.
    *   [Просмотр исходного кода](js/arrays.js) — Полная библиотека реализованных методов.
